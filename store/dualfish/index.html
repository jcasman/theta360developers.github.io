Dual-Fisheye Stitching

Dual-Fisheye Stitching Library and Application by Ichi Hirota

Sign Up to Receive Free Demo Version

Convert RICOH THETA dual-fisheye images taken with a THETA V plug-in into equirectangular images. This is an Android application for software developers to stitch a dual-fisheye image from the RICOH THETA V into an equirectangular image. The library comes with an Android calibration application that will store lens parameter information.

Calibration application is included with stitching library.
